# Compatibility

Target: Minecraft Beta 1.7.3 on Babric/StationAPI and Java 17. MultiMC is the supported manual launcher. The foundation is side-separated and writes no save data.

Build compatibility passed with Gradle 8.12.1, Fabric Loom 1.10.5, Babric Loom extension 1.10.5, mappings `b1.7.3+e1fe071`, Loader 0.19.3, and StationAPI 2.0.0-alpha.6.2. Client, dedicated-server, and MultiMC compatibility remain pending manual/runtime execution.
