# Compatibility

Target: Minecraft Beta 1.7.3 on Babric/StationAPI and Java 17. MultiMC is the supported manual launcher.

Build compatibility passed with Gradle 8.12.1, Fabric Loom 1.10.5, Babric Loom extension 1.10.5, mappings `b1.7.3+e1fe071`, Loader 0.18.4, and StationAPI 2.0.0-alpha.6.2. Babric Clean and StationAPI Baseline passed manually in MultiMC on Java 17. Both the Harvester client (singleplayer) and the Harvester dedicated server (multiplayer, server-authoritative) load, run the full harvest feature set, and shut down cleanly — validated by a full manual gate, not compilation alone.
